# Symtomsclassification

## 침윤성 유관암 이미지기반 CNN 예측모델

> **2023년 창의산출물 대회**
>
> **개발기간 : 2023.11 ~ 2023.12**

## 프로젝트 소개

**Tensorflow**를 이용하여 침윤성 유관암 이미지를 **CNN**모델에 학습 후 pos/neg 판별

**CNN모델 정확도 : 78%**
